# Spellcaster
Spellcaster是一款双人回合制战略棋盘游戏，考验玩家的战略思维和预判能力，游戏规则由Bartle在1977 年首次提出。详细游戏规则见该链接：
https://www.andrew.cmu.edu/user/gc00/reviews/spellcaster.html

本项目为该游戏的java实现，使用LibGDX引擎。

版本信息：
* JDK：openJDK17 
* gradle: 7.5.1
* LibGDX: 1.12.0
* Android SDK：34
